Loss formula (analytic_min_p_hinge)
Given next-token labels y_t (labels[:, 1:]) and unscaled logits l_t (logits[:, :-1, :]): valid_mask = (y_t != -100) logits_valid, labels_valid, temp_valid = l_t[valid_mask], y_t[valid_mask], T_t[valid_mask] If goldilocks_filter: Select a subset of tokens from (rank==1) and (rank<=topk, not easy) according to easy_frac/topk_frac. For each selected token: delta = relu(max(logits_valid) - logit_of_true_label) denom = -log(min_p_ratio) temp_lower_bound = delta / denom hinge = relu(temp_lower_bound - temp_valid) token_loss = temp_hinge_weight * hinge + temp_reg_weight * relu(temp_valid - temp_lower_bound)^2 temp_loss = mean(token_loss) loss = temp_loss * temp_loss_weight
